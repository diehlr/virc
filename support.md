## Reporting bugs

If you've found a bug, please email me with:

- What version of v.irc you're running (check the About menu)
- What version of macOS you're running
- A description of what went wrong
- If possible, steps to reproduce the issue

## Frequently asked questions

**Does v.irc work with IRC network X?**
v.irc supports standard IRC servers. If you can connect to a network with other IRC clients using the IRCv3 protocol, v.irc should work with it too.

**Is my message history stored in the cloud?**
No. All data stays on your Mac. See the [privacy policy](privacy) for details.

**How do I update v.irc?**
If you installed from the Mac App Store, updates come through the App Store automatically.
